# headlessdev Token Benchmarks

Per-page-type reduction range — reproducible. Run `pnpm bench` to verify.

| Page Type | Typical Raw | Typical Distilled | Reduction Range |
|-----------|-------------|-------------------|-----------------|
| API docs | 60-90k tokens | 1.5-3k tokens | 30-50x |
| Docs landing | 25-40k tokens | 2-4k tokens | 8-15x |
| E-commerce PDP | 50-70k tokens | 3-5k tokens | 12-20x |
| Blog post | 15-25k tokens | 3-5k tokens | 3-6x |
| SPA dashboard | 30-50k tokens | 2-4k tokens | 10-20x |

We publish the low end because trust matters more than headlines.
